# Plantilla Salud y Bienestar

Clínica de salud y bienestar profesional. Diseño profesional, moderno y responsive.

## Personalización
Edita las variables en `style.css`:
```css
:root {
    --color-primary: #2e86ab;
    --color-secondary: #1a1a2e;
    --color-accent: #a8d8ea;
}
```
Actualiza textos, imágenes y WhatsApp en `index.html` y `script.js`.
