# Plantilla Salud y Bienestar

Plantilla web profesional para clínicas, consultas y centros de salud.

## Negocios ideales
Dentistas, fisioterapeutas, psicólogos, nutricionistas, podólogos, clínicas estéticas, logopedas, entrenadores personales, pilates, yoga, quiroprácticos.

## Secciones
Hero profesional | Sobre la clínica | Servicios/Tratamientos | Beneficios | Equipo médico | Testimonios | FAQ accordion | Formulario de cita | Horarios y mapa | WhatsApp flotante

## Personalizar
Edita variables CSS en `style.css` (:root). Modifica textos en `index.html`. Cambia WhatsApp en `index.html` y `script.js`.

## Despliegue
Sube a cualquier hosting o activa GitHub Pages.
